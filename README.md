# tkinter-template
custom tkinter gui template application for cross platform python applications. Comes with build scripts for each OS system.
If you are downloading from this repository and not from my website, you must have py2app installed, and you have to uninstall pyinstaller from your global python environment if it already is.

### how to check
1. `pip3 freeze` <br>
2. `pip3 uninstall pyinstaller` <br>
